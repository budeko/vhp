# VHP Panel

Bu reponun resmi panel uygulaması. Kurulum yapan herkes varsayılan olarak bu paneli kullanır.

- **Geliştirme:** `npm install && npm start` (port 3000)
- **Image:** `docker build -t vhp-panel .` (veya repo kökünden `./scripts/build-images.sh`)
- Health: `/health`, `/ready`

Gerçek arayüz ve API bu minimal sunucu üzerine eklenecek.
